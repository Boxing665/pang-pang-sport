## 🎯 主要功能

### 1️⃣ 運動類型切換 (新功能！)

應用現在支持三種運動的實時篩選：

```
┌─────────────────────────────────────┐
│  PANG PANG SPORTS 預測        [🌐]  │
├─────────────────────────────────────┤
│  [全部] [⚽足球] [🏀籃球] [⚾棒球]  │
│                                     │
│  [📋 模擬數據]                      │
├─────────────────────────────────────┤
│                                     │
│  ⚽ 英超 | Today 18:00            │
│  ┌─────────────────────────────┐   │
│  │ 曼城        2 : 1    利物浦  │   │
│  │ ===我們的預測=== ✅ 72%      │   │
│  │ [主2.08] [和3.35] [客3.10]  │   │
│  │ 📌 曼城市場隱含勝率較佳      │   │
│  └─────────────────────────────┘   │
│                                     │
│  ⚽ 西甲 | Today 19:30            │
│  ┌─────────────────────────────┐   │
│  │ 皇馬        1 : 1    巴薩   │   │
│  │ ═══════════预测════════════ ✅65%│
│  │ [主1.95] [和3.50] [客2.15]  │   │
│  │ 📌 两队实力接近，平局风险    │   │
│  └─────────────────────────────┘   │
```

### 2️⃣ 如何切換運動

**跟隨以下步驟：**

1. **查看所有賽事**
   - 應用啟動時顯示全部運動的比賽

2. **點擊運動標籤**
   - `[⚽足球]` → 只看足球比賽
   - `[🏀籃球]` → 只看籃球比賽
   - `[⚾棒球]` → 只看棒球比賽
   - `[全部]` → 回到全部運動

3. **見證即時篩選**
   - 列表會立即更新
   - 只顯示選定運動的賽事

### 3️⃣ 預測信息

每個比賽卡片包含：

| 項目 | 說明 | 範例 |
|------|------|------|
| 🕐 **時間** | 比賽開始時間 | Today 18:00 |
| ⚽ **聯賽** | 運動 + 聯賽名 | ⚽ 英超 |
| 🏆 **預測比分** | AI 預測結果 | 2 : 1 |
| 📊 **信心度** | 預測可信度 | ✅ 72% |
| 💰 **賠率** | 國際賭盤賠率 | 2.08 / 3.35 / 3.10 |
| 📌 **關鍵因素** | 預測說明 | 主隊市場隱含勝率較佳 |

### 4️⃣ 數據源切換

**右上角圖標含義：**

- 🌐 **綠色** = 真實數據
  - 需配置 API keys
  - 從國際賭盤抓取即時數據

- 📋 **橙色** = 模擬數據 (預設)
  - 內置測試數據
  - 立即加載，不需網絡

**切換方法：**
- 點擊右上角的數據源圖標進行切換

## 💻 技術亮點

### 🏗️ 架構
```
多層數據管理系統
├── 統一入口: DataServiceManager
├── 模擬層: MockDataService
└── 真實層: RealDataService
    ├── TheOddsAPI (賠率)
    └── ESPN API (賽事)
```

### 🧠 預測引擎
```
EnhancedPredictionEngine 計算因子
├── 市場隱含概率 (60%)
├── 球隊現況分析 (20%)
├── 傷停影響調整 (10%)
└── 信心度評分 (50-92%)
```

### 🎨 UI 層級
```
自適應佈局
├── 運動篩選器 (橫向滾動)
├── 數據源指示器 (實時反饋)
└── 動態比賽列表 (即時篩選)
```

## 🎮 互動指南

### 場景 1: 只看足球
```
1. 應用打開，顯示所有運動
2. 點擊 [⚽足球] 籌選項
3. 列表自動更新，只顯示足球比賽
```

### 場景 2: 檢查籃球預測
```
1. 點擊 [🏀籃球] 篩選項
2. 查看今天的 NBA 比賽
3. 點擊任意比賽卡片查看詳細分析
```

### 場景 3: 刷新數據
```
1. 向下拉動列表（任意籽選狀態）
2. 等待圓形加載圖標完成
3. 數據將更新
```

### 場景 4: 切換數據源
```
1. 點擊右上角 [🌐] 或 [📋]
2. 數據源圖標顏色改變
3. 列表數據可能更新（如已配置真實 API）
```

## 🐛 故障排除

### 問題: 篩選不工作
**解決:**
1. 確認已點擊正確的運動標籤
2. 檢查是否有該運動的賽事
3. 嘗試刷新頁面

### 問題: 數據沒有更新
**解決:**
1. 查看左上角的數據源指示器
2. 確認使用的是模擬還是真實數據
3. 向下拉動進行手動刷新

### 問題: 應用崩潰
**解決:**
1. 在 VS Code 中按 R 進行熱重啟
2. 若無效則重新啟動應用
