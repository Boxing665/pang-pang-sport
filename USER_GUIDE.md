# 體育賽事預測應用 - 使用指南

## 🎯 功能特色

### 運動類型切換
應用支持三種主流男性體育賽事：

| 運動 | 圖標 | 支持的聯賽 |
|------|------|----------|
| ⚽ **足球** | ⚽ | 英超、西甲、意甲、日職等 |
| 🏀 **籃球** | 🏀 | NBA、歐籃等 |
| ⚾ **棒球** | ⚾ | MLB、中華職棒等 |

## 📱 UI 組件說明

### 1. 頂部導航欄
```
[PANG PANG SPORTS 預測] [🌐/📋]
```
- **左側**: 應用標題
- **右側圖標**: 切換數據源
  - 🌐 綠色 = 真實數據 (需配置 API keys)
  - 📋 橙色 = 模擬數據 (測試用)

### 2. 數據源指示器
```
[🌐 真實數據] 或 [📋 模擬數據]
```
顯示當前使用的數據源狀態

### 3. 運動類型篩選器
```
[全部] [⚽ 足球] [🏀 籃球] [⚾ 棒球]
```
點擊任意選項篩選該運動的賽事

### 4. 比賽卡片
每個卡片包含：
- **聯賽信息**: `⚽ 英超`，開賽時間
- **隊伍對比**: `主隊 vs 客隊`
- **預測比分**: 中央顯示 `2 - 1`
- **信心度條**: 綠/黃/橙 表示預測信心
- **賠率**: 主勝/和/客勝 賠率
- **信心度百分比**: `72%` 等
- **關鍵因素**: 簡短分析文本

## 🎮 交互操作

### 切換運動類型
1. 向上滑動查看運動篩選器
2. 點擊目標運動（例如「⚽ 足球」）
3. 列表會立即篩選，只顯示該運動的賽事
4. 再次點擊相同運動可返回全部

### 刷新數據
1. 向下拉動列表進行整頁刷新
2. 加載中會顯示旋轉圖標

### 切換數據源
1. 點擊右上角的數據源圖標
2. 應用會在真實數據和模擬數據間切換
3. 顏色會改變以反映當前狀態

## 📊 預測信息解讀

### 信心度顯示 (Confidence)
- 🟢 **78% 以上** = 高信心 (綠色)
- 🟡 **66-77%** = 中高信心 (黃色)
- 🟠 **50-65%** = 保守觀察 (橙色)

### 預測比分
- 基於球隊形態、賠率、傷停情況
- 為平衡各隊實力的理論預測
- 不代表實際比賽結果

### 賠率面板
- **主勝** (Home Win): 主隊獲勝的賠率
- **和** (Draw): 平局賠率（籃球/棒球無此項）
- **客勝** (Away Win): 客隊獲勝的賠率

## 🔄 數據更新

### 模擬數據模式 (預設)
- 使用內置 mock 數據
- 立即加載，無網絡延遲
- 適合測試功能

### 真實數據模式
- 需配置 API keys (見 REAL_DATA_SETUP.md)
- 從 TheOddsAPI 和 ESPN 獲取實時數據
- 約 10-30 秒加載時間（取決於網絡）

## 💡 最佳實踐

1. **先用模擬數據測試**: 確保功能正常後再配置真實 API
2. **注意 API 限額**: 免費 API 有月度限制
3. **檢查時區**: 比賽時間基於系統設置
4. **定期刷新**: 點擊運動類型時自動重新篩選

## 🐛 常見問題

### Q: 為什麼某個運動類型頁面顯示「該運動暫無賽事」？
A: 當前沒有該運動的預定賽事。稍後再試或查看其他運動類型。

### Q: 如何只查看特定聯賽？
A: 目前應用按運動類型篩選。如需按聯賽篩選，請等待未來更新。

### Q: 預測信息為什麼改變了？
A: 如果切換了數據源或刷新頁面，數據會更新。預測是實時計算的。

### Q: 籃球和棒球為什麼沒有「和」選項？
A: 這些運動沒有平局結果，所以賠率不提供此選項。
