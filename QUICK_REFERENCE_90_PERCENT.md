## 🔍 故障排除

### 问题 1：置信度仍然很低（<75%）
```
原因可能：
❌ 数据不完整（伤兵列表不全）
❌ 两队实力接近（市场模糊）
❌ 重要数据缺失（天气、关键球员）

解决方案：
✅ 检查数据来源 (ESPN API, Bet365)
✅ 等待更多信息 (比赛前 6 小时更新)
✅ 跳过此比赛，选择更明确的对手
```

### 问题 2：置信度过高（>95%）
```
原因可能：
❌ 数据质量问题（过时信息）
❌ 市场共识过度（可能有冷门）
❌ 权重配置不当

解决方案：
✅ 验证数据时间戳
✅ 检查赔率运动 (反向盘口信号)
✅ 查看多个博彩平台的赔率一致性
✅ 对极度自信的预测设置上限 (95%)
```

### 问题 3：准确率没有改进
```
原因可能：
❌ 参数权重不适合你的数据
❌ 数据质量有问题
❌ 样本量太小（<100 场）

解决方案：
✅ 运行 100+ 场回测
✅ 逐项验证 17 个特征的有效性
✅ 调整权重（高权重给最相关特征）
✅ 添加更多数据来源
```

## 💬 常见问题 FAQ

**Q: 这个系统能保证 100% 准确率吗?**
A: 不能。但它能让你的正确预测达到 90%+ 置信度，同时避免过度自信。目标是准确率与置信度匹配。

**Q: 集成需要多长时间?**
A: 5 分钟。只需 6 行代码（2 行导入 + 3 行调用 + 1 行配置）。

**Q: 是否支持其他运动（如冰球、美式足球）?**
A: 当前支持足球、篮球、棒球。扩展其他运动需要添加对应的 Enhancer 类。

**Q: 如何调整参数以优化准确率?**
A: 
1. 运行 100+ 场回测
2. 分析哪个特征贡献度最高
3. 增加高贡献特征的权重
4. 减少低贡献特征的权重
5. 重新回测验证效果

**Q: 置信度突然下降是什么原因?**
A: 
- 市场观点改变（赔率波动）
- 关键数据更新（伤兵、球队状态）
- 季节变化（冬季 vs 夏季）
- 数据源问题（API 延迟）
